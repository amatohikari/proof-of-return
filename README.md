# Proof of Return

**Restoring Equivalent Exchange: A Reductive Currency System**

*All things are impermanent — everything arises, exists, and returns.*

---

## Abstract

Everything that humans exchange daily — labor, time, meals, services — is finite and returns the moment it is consumed. Yet the medium designed to exchange these things — currency — has been engineered not to return. This asymmetry has bred hoarding, obstructed circulation, and entrenched a structure in which only those closest to the faucet benefit (the Cantillon effect).

This paper proposes a currency system that resolves this contradiction by embedding *return* into currency itself.

Satoshi Nakamoto demonstrated through Proof of Work "a method for reaching consensus without a trusted third party." This paper demonstrates through Proof of Return "a method for exchanging things that return, with a medium that also returns."

This system is named **AMI** — *All Material is Impermanent*.

## System Architecture

The system comprises three currencies — their initials form A, B, C:

- **Amortization** — Currency returned through exponential decay. All received currency decays to a 3% residual rate over 108 days, mirroring natural return rates (ash, decomposition, thermal cooling).
- **Base** — Existence guarantee currency. Each participant holds up to 1,000 Base (= 1,000 USDT). Used Base recovers exponentially at 6% daily — faster when low, slower near the ceiling.
- **Creation** — Creative support currency. Activated at 10 million participants. Grants up to 100,000 Base equivalent annually for education, research, and creation. Evaluated by participants and Decentralized AI.

The entire system is described by just two exponential functions — the decay of Amortization and the recovery of Base. No conditional logic. No human intervention. Autonomous equilibrium through natural law.

## Core Equations

**Decay (Amortization):**
```
Balance(t+1) = Balance(t) × (1 − r)
```
where r yields a 3% residual at T = 108 days.

**Recovery (Base):**
```
Recovery = (Deposit − Current Balance) × 0.06
```
Decay and recovery are mirror images of the same exponential function.

## Key Properties

- **No hoarding** — Currency decays; storing it is meaningless.
- **No Cantillon effect** — Base guarantees equal recovery for every participant.
- **No central bank** — Exponential functions maintain equilibrium autonomously.
- **Full recovery from zero** — Even after total loss, Base regenerates. No permanent death.
- **Currency euthanasia** — As the system matures, participants stop needing currency. This is the intended end state.

## Implementation

- **Initial L2:** Base network (Ethereum L2)
- **Final form:** Independent L2 inheriting Ethereum's security
- **Smart contract:** Lazy evaluation — decay is calculated at the moment of query, not per-second
- **Entry:** USDT/USDC converted to Base; stablecoins stored in Stablecoin Pool (SCP)
